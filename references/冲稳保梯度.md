# 冲稳保梯度

> 适用范围：梯度思想**全国通用**；但"可填几个、各档填几个"**因省而异**（可填志愿数差异极大）。
> 软知识库（L0）。讲方法，不替考生决策；本工具 Q1 的分档算法与本文一致（见 DESIGN §5）。

## 一句话

把有限的志愿名额按"录取可能性从低到高"排成**梯度**：少量**冲**（往年比你略高、有希望够一够）、中间**稳**（位次相当、大概率录）、压底**保**（往年明显低于你、几乎稳妥）。

## 三档怎么界定（本工具用法）

以**位次**为准（位次年际可比，分数不可比）。本工具用「近3年录取位次**均值** r_avg」与考生位次 rank 比，按差值比分档：

| 档位 | 含义 | 本工具阈值（差值比 = (r_avg − rank)/rank） |
|---|---|---|
| **冲** | 院校往年比你略高，够一够有希望 | −0.20 ≤ 差值比 < −0.08 |
| **稳** | 位次相当，大概率录 | −0.08 ≤ 差值比 < +0.10 |
| **保** | 你明显优于院校往年，几乎稳妥 | 差值比 ≥ +0.10 |

> 为什么用"近3年均值"而不是"去年最低分"：单年最低位次常是压线捡漏的**离群值**，且高校录取有**大小年**波动。用均值更稳，本工具同时展示"近3年位次区间"让你看到波动幅度。

## 怎么排梯度（防滑档的关键）

1. **保底必须绝对安全、宁稳勿空**：保的院校往年位次要明显低于你，且**不止一个**。这是不滑档的底线。
2. **拉开梯度**：冲、稳、保之间留出位次差，别全挤在同一段。
3. **冲要想清楚调剂**：冲的志愿被提档后若专业满了又不服从调剂 → 退档（见 [滑档与退档.md](./滑档与退档.md)）。
4. **参考配额**：本工具按你所在省的可填志愿数给"冲约20% / 稳约50% / 保约30%"的建议数量，仅供参考。

## 因省而异（必查具体省份）

可填志愿数量差异巨大，直接决定每档填几个：

| 省份示例 | 投档单位 | 本科批可填数（示意，以当年官方为准） |
|---|---|---|
| 传统文理省 | 院校 | 几个院校志愿 |
| 湖北/江苏/北京等 | 院校专业组 | 数十个 |
| 浙江 | 专业(类)+院校 | 80 个 |
| 山东 | 专业(类)+院校 | 96 个 |

→ 具体数量与投档单位以考生**所在省份**当年规则为准，本工具读 `data/provinces/<省>/时间线.json` 的"可填志愿数"。
