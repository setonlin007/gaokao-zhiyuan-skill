# 高考志愿 · 诚实版 AI 助手（gaokao-zhiyuan）

> **把公开的高考数据，整理到每条可回溯、数字不幻觉、算法看得懂、不承诺概率、不卖焦虑。**
> 一个负责任的"信息整理"工具（Claude Code skill）——不替你决定，但把决定需要的事实，老老实实摆给你看。

`数字不经 AI 生成` · `点级可回溯到考试院原文` · `算法全透明` · `绝不输出"录取概率%"` · `自检全过`

---

## 为什么做这个

每年六月，志愿填报是一场信息不对称的硬仗。市面上的 AI 志愿工具，几乎都栽在同样几个坑里：

- **数字是 AI 编的**——大模型从搜索摘要里"抽"数字，实测出现过"把黑龙江 4000 名考生推荐清华北大"这种硬伤；
- **"录取概率 87%" 不可信**——同一个分数，不同 App 给出不同学校、不同概率，专家明说"仅凭大数据无法判断能否录取"；
- **"独家数据 / 官方授权" 多是话术**——福建省考试院已公开澄清官方录取数据从未接入任何第三方 App；数据本就对所有人公开；
- **真正的钱在线下高价一对一**——App 只是引流，焦虑才是商品。

教育部已多次发文警示志愿填报 App 乱象。**这个项目反着来：行业的每一个软肋，我们逐条做成卖点。**

| 行业软肋 | 我们的做法 | 一句话 |
|---|---|---|
| 数字幻觉 / 失实 | 硬数字**确定性查表**，AI 只把结果翻译成人话 | "每个数字都来自考试院原件，**不是 AI 编的**" |
| 来源只是品牌口号 | 每条硬数据带 `source_url + 年份`，原件归档 + 哈希 | "每条数据**可点回官方出处**核对" |
| 黑箱 "AI + 大数据" | 冲稳保规则、阈值、分档逻辑全部写在明处 | "**算法看得懂**，你知道它怎么算的" |
| 概率不可信 | **绝不输出 "录取概率 X%"**，只给近 3 年位次区间 + 冲稳保分档 | "不骗你 '87% 录取'，只给**可核对的位次事实**" |
| "独家 / 保录取" 卖焦虑 | 不卖独家、不承诺录取、不替你决定 | "公开数据平权，我们卖的是**准、可验证**" |
| 忽略时点与省差异 | 阶段感知（考前/考中/考后）+ 省差异必查 | "同一问题**考前考后答案不同**，且**按你的省**算" |

## 它能帮你做什么

### 🟢 看规则 · 一次讲明白（零数据风险）
平行志愿到底怎么投档？滑档和退档差在哪？冲稳保该怎么排梯度？学校、专业、城市到底先选哪个？
→ 给你权威科普 + 例子 + 决策框架，**讲清逻辑，不替你拍板**。

### 🔵 看校 / 看专业 · 用扎实数据
- **优势学科**：基于教育部双一流建设学科名单（436 条）+ 全国高校库（2869 所），告诉你某校真正的王牌学科、城市、办学层次、性质；非双一流如实说"未列入"，不吹。
- **专业冷热**：麦可思红黄绿牌（🟢绿牌就业较好 / 🔴红牌预警，2025 届联网双源核实）——明确标注**第三方·参考·非官方，绝不据此断定某专业不能报**。
- **院校就业**：广东 15 所重点院校《毕业生就业质量年度报告》官方指针，8 所带可回溯的毕业去向落实率，其余只给官方报告入口，**绝不预填未确证的数字**。

### 🔴 选校 · 冲稳保候选清单（核心能力，广东已落地）
给定**分数（或位次）+ 选科**，输出冲 / 稳 / 保三档候选投档单位，每一条都带：

```
🔴 冲
  · 深圳大学（组233）[深圳]  位次11801(2025)  [选科✅全资格]  扩招  💼就业86.4%·2024届
  · 华南理工大学（组214）[广州]  位次11609(2025)  [选科✅全资格]  ↑近年更难  🎓双一流:化学、材料科学与工程…  💼官方就业报告·2024届
```

一行之内，你同时看到：**院校专业组**粒度的录取位次 + 年份、**选科是否有资格**、**双一流学科**、**大小年冷热趋势**、**扩缩招**、**就业落实率**——还有梯度配额建议（冲几个/稳几个/保几个）、调剂退档风险提示、以及每条数据的**官方来源链接**。

> **没有"录取概率 87%"，只有"近 3 年位次事实 + 冲稳保分档"。** 决定权始终在你手里。

## 它最硬核的两件事

### 1. 选科是"真过滤"，不是摆设
新高考省最致命的坑是"够得上 ≠ 有资格"——你选了物化生，清单里却混进一堆要求政治的专业。本工具做了**院校×专业类级真过滤**：
- 凡"本轨专业类你全部无资格"的院校，在排位次之前就 **hard-exclude**（实测一个缺化学的考生，34 所整校无资格院校被直接剔除）；
- 每个候选标注"该校你有资格 N/M 个专业类"并点名无资格的专业；
- 诚实声明数据天花板：受限于官方"专业组→专业"映射尚未公开，组代码级的精确剔除留待官方招生专业目录到位——**宁可说"待核对"，也不假装已过滤**。

### 2. 抗大小年，用的是对的方法
实测广东 2023–2025，专业组"组号年年重组"，三年只有约 12% 能按组号对齐——所以本工具**不做行业流行的"近 3 年位次均值"**（那会把重组后的不同组张冠李戴），而是：组级取最新年为基准，**抗大小年改由院校级近 3 年趋势**（实测 89% 院校稳定）+ 招生计划增减提示。这是趴在数据上啃出来的修正。

## 数据从哪来，凭什么可信

所有硬数据只来自**官方/权威公开源**，每行强制带 `source_url`、`year`、`fetched_at`、原件哈希：

- 广东省教育考试院：一分一段表、投档线、招生计划、官方选考科目要求查询系统；
- 教育部：双一流学科名单、全国高校名单；
- 麦可思研究院：红黄绿牌专业（第三方·半硬数据·参考）；
- 各高校信息公开网：毕业生就业质量年度报告（官方报告指针）。

> 第三方聚合 / 商业 API **仅作 URL 发现线索，绝不进库**。建库走季节性快照管线：声明式源注册 → allowlist 官方域抓取并归档原件 → **确定性解析（按列取值，数字不经 LLM）** → 自动校验闸门 → 留痕版本化。

**数据规模一览**：广东投档单位最低位次 13,990 行（2023/2024/2025 三年）· 一分一段 1,171 行 · 选科要求 83,096 行 · 双一流学科 436 条 · 全国院校库 2,869 所。

## 架构（分层，硬软分离）

```
L3 输出层   带来源 + 阶段口径 + 免责的结构化答案
L2 能力层   P0 阶段判定 → Q0 科普 / Q2 看校 / Q1 选校
L1 逻辑层   阶段判定 / 冲稳保分档 / 位次换算 / 选科过滤   ← 确定性，无 LLM 生成数字
L0 数据层   硬数据(位次/计划/选科/名单) + 半硬数据(就业) + 软知识库，均标来源年份
```

- **硬数据**（位次/计划/选科）：确定性查表，错一个数字就是误导，绝不经 LLM。
- **半硬数据**（就业质量/红黄绿牌）：打包并标来源年份，可降级为软信息。
- **软信息**（专业是干嘛的/城市印象）：才交给 AI + 联网，且与硬数据分开标注。

## 边界与免责

- **不预测、不承诺、不替你决定**：不算"你能发挥多少分"，不给"录取概率%"，不说"保录取/不滑档"，不替你选学校。
- **不卖独家**：数据本就公开，价值在"整理得准、可验证"。
- **省差异必查**：志愿模式 / 可填数量 / 批次 / 命题各省不同，绝不写成全国通则。
- **数据有时效**：出分前只能用往年数据做估算（会明确标注"估算·以当年为准"）；一切以各省考试院最新发布为准。

> 本工具由 AI 整理公开信息，仅供参考，**不构成填报建议或录取承诺**。数据请以各省考试院最新发布为准。
