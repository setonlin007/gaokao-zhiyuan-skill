# data/employment/ — 就业（半硬数据）

> **半硬数据**（DESIGN 铁律二）：有权威可打包来源，但容错性高于投档线/位次。
> **必须标来源年份 + 数据性质**；缺数据可降级为软信息；**不当作官方硬数据、不决定报考资格**。

## 文件

- `红黄绿牌专业.csv` —— 麦可思研究院《中国本科生就业报告(就业蓝皮书)》的本科红/绿牌专业。
  - **绿牌**：就业需求增长、薪资与满意度较高的专业；**红牌**：失业量较大、就业率/满意度较低、持续预警的专业。
  - **数据性质：第三方研究机构(麦可思)，非官方**。仅供"专业冷热"参考，**就业好坏因人、因校、因地而异，绝不据此断定某专业不能报**。
  - 列：`专业 / 牌色 / 口径 / year / 发布机构 / 数据性质 / source_url / fetched_at`。
  - **2025届(2025年6月11日发布)联网确证名单**：
    - 红牌(5)：公共事业管理、音乐表演、绘画、法学、美术学（法学/绘画连续5年红牌；公共事业管理首次上榜）。
    - 绿牌(6)：电气工程及其自动化、微电子科学与工程、机械电子工程、新能源科学与工程、车辆工程、机器人工程（全部为工科；电气连续多年登顶；车辆工程首次上榜）。
    - 来源：人民政协网/澎湃新闻转载麦可思2025年版就业蓝皮书（见 CSV 内 source_url），交叉一致。
  - **诚实说明（口径降级）**：
    - 旧占位 CSV 里标为 2024 届绿牌的「信息工程/信息安全/信息网络/数字媒体技术」**未能联网确证**，且多个来源指出信息安全/网络工程/软件工程等"近三年未进入绿牌榜单"，故**已移除**，不保留无来源条目。
    - 「应用心理学」近5年5次上榜4次为高频红牌（确证），但其是否进入 2025 届当年红牌5强未单独确证，口径已在 CSV 中注明。
    - 「能源与动力工程」按 2024 届/近5年高频绿牌保留，来源为皮书网转载麦可思 2024 蓝皮书；非 2025 届当年绿牌6强。

- `院校就业质量指针.csv` —— 广东 15 所主要本科院校《毕业生就业质量年度报告》的**官方报告指针**（report pointer）。
  - **定位**：不预填未确证的就业率/薪资数字；只给到**校方信息公开网/就业中心的官方报告入口 URL** + 可确证的少量摘要，让 skill 把家长导向官方原文。这是"宁缺毋滥"的诚实降级。
  - 列：`院校名 / 报告名 / year / 关键指标摘要 / source_url / fetched_at / 数据性质`。
  - **已收录 15 所**：中山大学、华南理工大学、暨南大学、深圳大学、华南师范大学、华南农业大学、广东工业大学、南方科技大学、广州大学、汕头大学、广东外语外贸大学、南方医科大学、广东财经大学、东莞理工学院、广州医科大学。URL 均为各校**信息公开网/就业中心官方栏目页或官方报告页**（非第三方转载）。
  - **已确证「毕业去向落实率」数字（标了届数；精确值仍以官网原文为准）**：
    - 南方科技大学 2023届 总体约95.82%、本科约94.99%
    - 广东外语外贸大学 2022届 总体约95.59%、本科约95.57%（已确证较早届）
    - 暨南大学 2023届 约92.03%
    - 华南农业大学 2023届 总体约91.27%、本科约90.46%
    - 深圳大学 2024届 本科约86.4%
    - 广州大学 2024届 本科初次约85.78%（近3年本科约93%）
    - 汕头大学 2024届 本科总体约84.40%
    - 广东工业大学 2023届 给出去向构成（协议合同约53.31%/升学约21.34%），未取到单一总体落实率口径，存指针。
  - **仅存报告指针（未确证当年具体落实率）**：中山大学、华南师范大学、南方医科大学、广东财经大学、东莞理工学院、广州医科大学。
  - 摘要中数字多来自第三方对校方报告的转述（如华南理工 2024届本科约6526人），**精确值一律以官网原文为准**。

## 调用接口

- `lib/employment_lookup.py` —— 给 Q1/Q2 等调用方的纯函数层（无网络/无 LLM）：
  - `carded(专业名) -> {专业,牌色,口径,year,发布机构,数据性质,source_url} | None`（多条命中取 year 最新）
  - `school_employment(院校名) -> {院校名,报告名,year,摘要,source_url,数据性质} | None`
  - 名称做双向子串宽松匹配；CSV 缺失或无命中返回 None；返回项自带 source_url，调用方负责附"半硬·参考·非官方·不决定能否报考"免责。
  - `answer_q2.py` 已改为统一经此模块读取（单一事实源）。
